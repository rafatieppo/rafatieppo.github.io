lucyLattes (v1.0.0) um script para manipular dados da plataforma Lattes

extraindo dados do currículo Lattes

2024-08-17
datascience / lattes / lucyLattes / python / mineração / data mining

O que eu achava algo improvável aconteceu. Como outras pessoas estão utilizando o lucyLattes para analisar currículos, resolvi trabalhar em uma nova versão (v1.0.0), e aos poucos estou atualizando a nova versão.

Se estiver interessado na história do lucyLattes: https://rafatieppo.github.io/post/2019_03_13_lucylattes/ .

Na versão v1.0.0 , as melhorias são:

  • Reorganização do código: acredito estar melhor do que antes, pelo menos no que tange a estrutura, organização e por incrível que pareça, no código em si. Todavia, se que o código está longe de ser profissional. Ah! se você estiver se perguntando se foi realizado algum Processo de Teste, a resposta é não! Por que? Porque o código não tem erro! :) Além disso: Na minha máquina roda! Brincadeira a parte, no geral acho que está melhor do que antes.
  • O relatório tem pequenas melhorias, algumas explicações extras sobre as tabelas.
  • Há uma interface para fazer as configurações desejadas.
  • Para que usou a versão anterior, agora não é mais necessário ficar digitando ID de pesquisador(a) e nome no arquivo de texto.
  • E uma melhoria considerável (acho que foi a melhor), para excluir os mesmo artigos que aparecem para mais de um pesquisador foi implementando um algorítimo de NLP (Natural Language Processing). De modo resumido, foi utilizado a similaridade por cosseno para comparar os títulos. De um modo mais fancy, agora o lucyLattes tem I.A..
  • Deve ter mais coisas, mas agora não recordo.

Para instalar e usar, há vídeos no YouTube e mais informações no site do projeto (GitHub):

  • youtube: Rafael Tieppo .
  • GitHub: https://github.com/rafatieppo/lucylattes
