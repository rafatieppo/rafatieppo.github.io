Entendendo java (VM) para instalação do programa IRPF

Java (VM) para declararação IRPF

Introdução

Para fazer declararação de imposto de renda vc necessita reunir a documentação e instalar o programa IRPF. Para obter arquivo de instalação basta acessar a página da receita. O detalhe é que abaixo do link de download há uma mensagem com o seguinte aviso ao usuário:

ATENÇÃO! Caso necessário, atualize a versão JAVA neste computador. Por favor, clique aqui e prossiga com a instalação.

Dependendo de qual é o seu sistema operacional (Windows, Linux, Solaris, etc) a instalação do JAVA pode se tornar um pouco chata. Se você usar Windows, acredito que em condições normais você não terá maiores problemas. Pois basta acessar o site e seguir as instruções.

Caso você use Linux (meu caso), há mais de uma opção para instalação do JAVA, o que pode causar algumas confusões. Na tentativa de auxiliar na compreensão de como proceder a instalação do JAVA, resolvi escrever este post.

O que é JAVA?

Para deixar claro, vamos especificar quem é quem:

  • java script: é uma linguagem de programação mas é diferente de JAVA ;
  • JAVA: é uma linguagem de programação orientada a objetos;
  • Java Virtual Machine (JVM): para o português máquina virtual java

Um dos motivos que fez da linguagem de programação JAVA um sucesso é o seu conceito de multi-plataforma. Inclusive, um dos slogans do JAVA é “WORA” (Write once, run anywhere), que na tradução para o português significa: Escreva uma vez, execute em qualquer lugar.

Sendo extremamente simplista, na prática isto isto significa que uma vez que o desenvolvedor escreve o código em JAVA, e gera um determinado programa, independentemente do seu sistema operacional (Linux, Windows, MAC, etc), basta você possuir a JVM para poder rodar o programa no seu computador. Isto porque a JVM faz todo o trabalho de carregar as dependências necessárias para rodar o programa. Caso você não tenha entendido o conceito de máquina virtual, imagine a máquina virtual JAVA como sendo um mini computador dentro do seu computador. Este mini computador é que executa os programas JAVA.

Qual JAVA instalar para rodar o programa do IRPF?

Para ter a JVM em seu sistema é necessário instalar o JRE (JAVA Runtime Environment), que é gratuito, e permite ao usuário rodar aplicativos JAVA no seu computador.

Agora, para quem deseja desenvolver aplicações JAVA, é necessário instalar o JDK (JAVA Development Kit). No JDK está incluso o que é necessário para escrever aplicações, e inclusive o JRE.

Por via das dúvidas, usualmente eu instalo o JDK.

Como instalar o JAVA no Linux para rodar o programa IRPF

Há pelo menos duas opções de JAVA, uma que é proprietária, pertencente à Oracle, e outra que é de código aberto OpenJDK com licença GPLv2. Se o uso do JDK for para fins comerciais, recomenda-se a leitura das licenças para ambas as versões, mas para rodar o IRPF ambas podem ser utilizadas sem problemas (este é o nosso objetivo).

Usualmente, nos repositórios das distribuições Linux o JDK disponível é o OpenJDK, e para evitar a adição de repositórios é esta a versão que vamos instalar. A sua versão mais recente (2020-04-25) é a 11.

Agora que já sabemos quem é quem, vamos realizar a instalação. Nas distros Linux Debian, Ubuntu, Mint, PopOS e derivados, acesse o terminal e digite:

sudo apt-get install openjdk-11-jdk

Uma vez que você finalizou a instalação do JDK, vamos obter o programa do IRPF. Acesse a página da Receita, posteriormente acesse a seção do IRPF2020 e acesse a área de downloads. Agora, procure o quadro que tem o nome Computador, clique em Outros (Mac, Linux, Solaris). Na sequência clique sobre SOLARIS/ OUTROS (jar), por fim clique sobre Programa IRPF 2020 para iniciar o download.

Após a conclusão do download um arquivo com extensão .jar estará salvo no seu computador. Coloque este arquivo em uma pasta da sua preferência. Via Terminal acesse a referida pasta e digite:

java -jar IRPF2020v1.6.jar

Assim que o comando acima for executado, uma janela de instalação do programa IRPF será aberta. Siga as instruções até finalizar a instalação. Por padrão o programa é instalado em /home/NomeUsuario/ProgramasRFB/ (Figura 1).

Para executar o programa IRPF, basta acessar a sua pasta /home/NomeUsuario/, e localizar a pasta ProgramasRFB. Nesta pasta estará a(s) versão(s) do(s) programa(s) IRPF. Via terminal, acesse a versão desejada, por exemplo, IRPF 2020, e execute a linha de comando (Figura 2):

java -jar irpf.jar

Se tudo ocorreu bem, basta usar o programa IRPF. Há um vídeo no youtube que explica como acessar o programa IRPF (video aqui).
